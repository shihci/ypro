詳細介紹

最多支持16個讀卡器

  MB3180、MB3280 和MB3480 是標準的Modbus 網關，可進行Modbus TCP 和Modbus RTU/ASCII 協議之間的轉換。最多同時支持16 個Modbus TCP master，每個串口最多31 個RTU/ASCII slave。對於RTU/ASCII master，最多支持32 個TCP slave。


商品特色

標準Modbus 網絡集成

三款標準MGate™ 型號（MB3180、MB3280 和MB3480）專為了快速集成Modbus TCP 和RTU/ASCII 網絡而設計。這些型號可無縫集成Modbus 串口slave 設備至現有的Modbus TCP 網絡中，並且讓串口master 訪問Modbus TCP slave。MB3180、MB3280 和MB3480 提供更簡單、可定制的網絡集成解決方案，且幾乎可與任何Modbus 網絡兼容。

高密度、經濟高效的網關全

MGate™ MB3000 網關可以有效地將高密度Modbus 節點連接至同一網絡。MB3280 最多可管理62 個串口slave 節點，MB3480 最多可管理124 個串口slave 節點。每個RS-232/422/485 串口可為每個Modbus RTU 或Modbus ASCII 配置不同的操作命令和波特率，允許兩種類型的Modbus 網絡通過一個Modbus 網關集成至Modbus TCP 網絡。

自動設備路由，簡化配置

Moxa 的自動設備路由功能為需要配置大量Modbus 設備的工程師解決許多問題和不便。只需一鍵即可設置slave ID 路由表完成Modbus 網關配置，並自動檢測來自監控和數據採集(SCADA) 系統的Modbus 請求。通過消除手動創建slave ID 路由表的需要，自動設備路由功能為工程師節省了大量時間和成本。
